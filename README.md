# Amanuensis

a - MAN - you - EN - sis

## Description

Amanuensis is an local web proxy that intercepts and displays the contents of HTTP requests and responses.  It is intended to be used as a debugging tool for web developers.

## Running

```bash
./gradlew :app:run
```
